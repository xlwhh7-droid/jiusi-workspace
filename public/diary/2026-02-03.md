### OAuth token 同步命令

```bash
# 从本地同步到硅谷
scp -i ~/.ssh/[SSH密钥] ~/.openclaw/agents/main/agent/auth-profiles.json root@[硅谷服务器]:~/.openclaw/agents/main/agent/auth-profiles.json

# 重启 gateway 生效
ssh -i ~/.ssh/[SSH密钥] root@[硅谷服务器] 'source ~/.bashrc; export PATH="$HOME/.local/share/fnm:$PATH"; eval "$(fnm env)"; openclaw gateway restart'
```
